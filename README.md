# Tarifka - React Native Yemek Uygulaması

Bu proje Patika React Native kursu kapsamında yapılmış bir **mobil yemek uygulamasıdır**. Kullanıcılar kategorilere göz atabilir, yemekleri listeleyebilir ve yemek detaylarını inceleyebilir.

## Özellikler

- Kategorilere göre yemekleri listeleme
- Her yemek için detay sayfası (malzemeler, tarif vs.)
- API üzerinden dinamik veri çekme (TheMealDB)
- React Native `FlatList` kullanımı
- Navigasyon için React Navigation kullanımı

## Teknolojiler

- React Native
- TypeScript
- React Navigation
- Axios
- TheMealDB API
- React Native Hooks (custom `useFetch`)


## Kullanım
Uygulama açıldığında kategoriler listesi gelir.

Kategoriye tıklayarak o kategoriye ait yemekleri görebilirsin.

Her yemeğe tıklayarak detay sayfasına ulaşabilirsin.
